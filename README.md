A playful single-page marketing site for a fictional dog hangout app. Built with HTML, CSS, and vanilla JavaScript to showcase services, testimonials, and contact options for dog lovers.

## Features
- **Responsive layout** that adapts to phones, tablets, and desktops.
- **Hero, features, and CTA sections** encouraging visitors to join BarkBuddy.
- **Download badges** for App Store / Google Play (SVG assets included).
- **Contact form UI** handled by `contact.js`.
- **Reusable icons and illustrations** stored under `images/` plus custom SVGs.

## Getting Started
```bash
git clone https://github.com/<your-user>/BarkBuddy-Site.git
cd BarkBuddy-Site
```
Open `index.html` in your browser, or serve it locally:
```bash
npx serve .
```
